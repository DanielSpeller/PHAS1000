In circular coordinates the position of an object is given by
$$\underline{r}=R\hat{\underline{r}}=R(\cos (\theta) \underline{i}+\sin(\theta)\underline{j})$$
If distance $=\theta$
$$\omega=\frac{d\theta}{dt}$$
$$\alpha=\frac{d\omega}{dt}$$
