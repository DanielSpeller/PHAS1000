Differentiation rules

Power rule
$$y=x^n$$
$$y'=nx^n-1$$
Chain rule
$$y=f(g(x))$$
$$y'=g'(x)\cdot f'(x)$$

Product rule 
$$y=f(x)g(x)$$
$$y'=f(x)g'(x)+g(x)f'(x)$$
Integration rules
$$\int x^n = \frac{x^{n+1}}{n+1}+c$$
$$\int e^{ax}=\frac{e^{ax}}{a}+c$$
$$\int\frac{1}{x}=\ln x + c$$
$$\int\frac{f'(x)}{f(x)}=\ln x $$
